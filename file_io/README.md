# C - File I/O

## Description

This project explores file input/output operations in C using low-level system calls. Unlike standard library functions like printf or fopen, this project works directly with the Linux kernel through syscalls such as open, read, write, and close.

Through this project, we learn how file descriptors work, how to manage file permissions, and how to handle errors when working with files at the system level.
