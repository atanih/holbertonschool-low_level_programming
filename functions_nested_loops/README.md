C - Functions, nested loops
 
    Project Description

This project introduces the fundamental concepts of functions and nested loops in C programming language. Students will learn how to:

	∙	Write and use functions efficiently
	∙	Understand function prototypes and declarations
	∙	Work with nested loops for complex iterations
	∙	Manage variable scope (local and global)
	∙	Use static variables
	∙	Follow Betty style guide for code quality

    Learning Objectives

At the end of this project, you are expected to be able to explain to anyone, without the help of External:

    General

*What are nested loops and how to use them
*What is a function and how do you use functions
*What is the difference between a declaration and a definition of a function
*What is a prototype
*Scope of variables
*What are the gcc flags -Wall -Werror -pedantic -Wextra -std=gnu89
*What are header files and how to to use them with #include
